# sistema_bancario

Description. 
The package sistema_bancario is used to:
	Processing:
        - Create client
	- Create account
        - Make a deposit
        - Meka a Withdraw
        - View account statement

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install sistema_bancario

```bash
pip install sistema_bancario
```
